- Offline Assist panel

- Tags recommendation
- Level recommendation

- Batch Dashboard
- QA color status
- Ctrl+S save

## Run

```bash
pip install -r apps/requirements_dictionary_editor.txt
python apps/dictionary_editor_qt.py
```

## Workflow

1. Open Excel.
2. Set Batch Range.
3. Click Apply.
4. Use Assist All.
5. Review Example_KO manually.
6. Save and Next.
7. Run QA.
